# 스택과 큐
### 스택
후입선출<br>
push pop peek<br>
깊이 우선 탐색, 백트래킹에 효과적<br>
### 큐
선입선출<br>
add poll peek<br>
너비 우선 탐색에 효과적<br>
### 우선순위 큐
들어간 순서와 무관하게 우선순위에 따라 요소가 제거되는 큐<br>
정렬 기준 직접 정의 가능
# 정렬
### 버블 정렬
인접한 요소끼리 비교 후 swap 하는 방식 o(n^2)<br>
가장 큰 값이 뒤로 가면서 뒤에서부터 정렬 완료
###  선택 정렬
최솟값 또는 최댓값을 찾고, 남은 정렬 부분의 가장 앞에 있는 데이터와 swap o(n^2)<br>
앞에서부터 정렬

### 삽입 정렬
선택된 데이터를 현재 정렬된 데이터 범위 내의 적절한 위치에 삽입 o(n^2)<br>
이진 탐색을 사용하여 시간 복잡도 줄이기 가능

### 퀵 정렬
기준값(pivot)을 정해 해당 값보다 작은 데이터와 큰 데이터로 분류하는 것을 반복해 정렬 o(n log n)<br>

### 병합 정렬
분할 정복 방식을 사용해 데이터를 분할하고 분할한 집합을 합침 o(n log n)<br>

### 기수 정렬
값을 놓고 비교할 자릿수를 정한 다음 해당 자릿수만 비교 o(kn)<br>
0-9에 해당하는 10개의 큐를 만들고, 일의 자리 수를 기준으로 저장 후 pop<br>
이를 다시 십의 자리 수를 기준으로 큐에 넣고 다시 pop -> 정렬 완료<br>

# 탐색

### 깊이 우선 탐색
시작 노드에서 출발하여 탐색할 한 쪽 분기를 정하여 최대 깊이까지 탐색을 마친 후 다른 쪽 분기로 이동
